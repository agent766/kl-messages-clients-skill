## Format — règles strictes

- Aucun emoji Unicode
- Aucun em-dash `—`
- Un seul CTA principal
- Signature courte (cf. agent)

## Format — règles modulables (révisées après aspiration officielle Evaneos)

| Élément | Règle |
|---|---|
| Bullets | Tolérés pour listes administratives (next steps, jalons paiement). Le template 2.1 les utilise |
| Smileys ASCII (`;)`) | Le template 2.4 en contient. Tolérés selon agent. **Interdits pour Alison** |
| Exclamations | Très rares. **0 pour Alison**. Max 1 dans les autres voix |
| Gras de section | Sur emails très structurés uniquement (manuel dans l'éditeur cible, pas en Markdown) |
