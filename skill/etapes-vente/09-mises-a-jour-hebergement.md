# Étape 09 — Mises à jour hébergement

## Quand

Entre la confirmation d'acompte (08) et le pré-départ (10), si une réservation hôtelière connaît un changement (annulation hôtel, indisponibilité de dernière minute, surclassement, modification de chambre).

## Objectif

1. Communiquer le changement avec transparence
2. Présenter immédiatement la solution proposée (pas le problème seul)
3. Confirmer que le tarif et le standing sont maintenus (ou expliquer si écart)
4. Demander accord du client si standing différent

## Structure

1. Salutation + ton factuel
2. Annonce du changement (1-2 phrases)
3. Cause si pertinente (sans excuser ni dramatiser)
4. Solution proposée (hôtel de remplacement, dates ajustées)
5. Confirmation : standing maintenu, tarif inchangé OU écart à approuver
6. Demande d'accord ou validation
7. Sign-off

Longueur cible : 8 à 15 lignes.

## Variations plateforme

### Kilt & Licorne

- Direct, pose le problème + solution dans le même paragraphe
- Si Nicolas connaît personnellement l'hôtelier de remplacement, le mentionner ("j'ai parlé avec [Prénom] hier soir, il vous garde une chambre vue lac")

### Evaneos

- Pas de template officiel dédié
- Sobre, factuel, pas de "désolé" répété

### Destination-Écosse

- Marie prend le ton consultatif : explication un peu plus développée, options multiples si possible

## Pièges

- Annoncer le problème sans la solution → angoisse client
- Multiplier les excuses → fragilise la relation
- Cacher un écart de standing pour ne pas avoir à demander accord → perte de confiance si découvert sur place
- Modifier sans accord du client si l'écart dépasse une simple équivalence

## Référence Evaneos

- Pas de template officiel dédié
- Tradition agence : transparence totale + solution immédiate

## À enrichir

- Template-type pour les 3 cas les plus fréquents (annulation hôtel, surclassement gratuit, changement chambre famille)
- Critères pour distinguer "équivalent" (pas besoin d'accord) vs "différent" (besoin d'accord)
- Liste d'hôteliers partenaires de confiance par région pour solutions de repli
